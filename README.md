# Cleanvo Care Solution — Official Website

Production marketing website for **Cleanvo** — premium home hygiene products.

**Tagline:** Your Everyday Hygiene Partner  
**Stack:** HTML5 · CSS3 · Vanilla JavaScript (ES6+) · EmailJS (contact) · GitHub Pages

No frameworks. No backend. No e-commerce.

---

## Local preview

Open any HTML file in a browser, or serve the project root:

```bash
# Python
python -m http.server 5500

# Or VS Code / Cursor Live Server on index.html
```

Then visit `http://localhost:5500`.

- Incremental, deployable sprints only
