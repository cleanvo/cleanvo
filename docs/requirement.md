# CLEANVO WEBSITE - MASTER REQUIREMENT DOCUMENT

## Project Overview

You are an expert Senior Frontend Architect, UI/UX Designer, Brand Designer, and Software Engineer.

We are building the official website for **Cleanvo Care Solution**.

This is **NOT** a demo project.

This is the company's official production website.

The website must look premium, modern, trustworthy, responsive, and professional.

The website should feel comparable to premium consumer brands like Harpic, Lizol, Dettol Home, Domex, etc., while maintaining its own identity.

The website should never look like a template or a generic AI-generated website.
